# Roadmap

## v0.1

- Core VibeChat request executor
- Full-screen terminal UI
- Clipboard-first chatbot handoff
- Session history and usage tracking
- Public proof-of-concept documentation

## v0.2

- Config file support for reusable preferences
- Better diff previews and file-change summaries
- Smoother onboarding for first-time chatbot sessions
- More export and handoff formats

## v0.3

- Optional richer terminal visuals and demo assets
- Broader local tooling primitives for advanced workflows
- More guardrails around destructive operations
- Better collaboration paths for outside contributors
